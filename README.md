# Cognee MCP Implementation

This project contains a Docker-based implementation of Cognee with MCP support.

## Stack
- **Cognee API**: Core knowledge graph and memory service.
- **Cognee MCP**: Model Context Protocol server for Cognee.
- **Postgres**: Relational and Vector database (pgvector).
- **Neo4j**: Graph database.
- **Redis**: Caching.
- **Nginx**: Auth proxy.
